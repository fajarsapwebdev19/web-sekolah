Perintah Dasar Git

=> Menambahkan File Kedalam Staging
git add

jika sebuah folder maka perintahnya adalah
git add <nama folder>

jika sebuah file maka perintahnya adalah
git add <nama file>

jika ingin menambahkan semua file ke proses staging 
git add .

=> membuat sebuah pesan
git commit -m "pesan anda"

=> cek sebuah remote
git remote -v

=> push data (menambahkan sebuah data ke dalam github)
git push -u origin <branch-name> (master/main)

=> menambahkan remote ke dalam git
git remote add origin <url repo anda>

=> inisialisasi remote
git init

=> hapus data
git rm

=> ambil sebuah data yang diupdate oleh contributor
git pull origin <branch-name> (master/main)
